Design Philosophy
-----------------

sfs-settings was created with the following principles in mind:

1. **Simplicity first**: Common operations should be easy and intuitive
2. **Flexible patterns**: Support multiple ways of managing settings to fit various use cases
3. **Security by design**: Proper handling of secrets with secure defaults
4. **Progressive disclosure**: Simple for basic use cases, powerful for complex ones

This guide explains the core concepts behind sfs-settings and how they work together.
